Boost.Mixin

Динамично композиране и мутиране на типове в С++

Автор Борислав Станимиров

Динамично кво?!

Композиране

  • Имаме парчета от типове - миксини
  • Например имаме класовете:
    • apple_product
    • mp3_player
    • headphones_player
  • Сглобяваме си iPod

Мутиране

Подменяме mp3_player с casette_player и получаваме iWalkman

Или...

подменяме apple_product с microsoft_product и получаваме Zune


Zune. Zune. Zune.

Или...

Добавяме, махаме, подменяме неща колкото ни душа иска

Миксин?

  • Най-много прилича на миксините в Ruby
  • Не е CRTP миксин
  • По-малко прилича на traits в Self, Scala, PHP, и др.
  • Съвсем малко на множествено наследяване

Приятности

  • Дъъ. Рънтайм типове
  • PIMPL
  • Неинтрузивна
  • Мултикасти (няма в Ruby)
  • Екстри. Алокатори. Комбинатори. Още екстри

Неприятности

  • Няма compile-time тип
  • Няма обект.метод(), а има метод(обект)
  • Харчи повече памет
  • Харчи повече ток —
    call overhead (пренебрежим (?))

Бам! Код


object obj;

mutate(obj)
    .add<apple_product>()
    .add<mp3_player>()
    .add<headphones_player>();

get_looks(obj); // -> "cool"
set_audio(obj, "some.mp3"); // handled by mp3_player
play(obj); // playing some.mp3 through headphones

mutate(obj)
    .remove<headphones_player>()
    .add<speakers_player>();

play(obj); // playing some.mp3 through speakers
